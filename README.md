# 🛹 Sk8erWise  
### The Skater Travel Guide

Sk8erWise is a digital discovery platform designed for the global skating community.  
Think **travel magazine meets interactive skate atlas** — a place where skaters can explore parks, plan trips, and share local knowledge.

---

## 🌍 What Is Sk8erWise?
Sk8erWise helps skaters discover the best skate parks around the world through a clean, card‑based interface.  
Each park is presented as a **destination card** with photos, location, and a short teaser.  
Users can filter parks, open detailed park info, save favourites, and read community reviews.

---

## 👥 Target Users
- **Traveling Skaters** – exploring new cities and parks  
- **Spot Curators** – sharing and recommending favourite parks  
- **Skate Explorers** – planning skate‑focused trips  
- **Local Guides** – showcasing their city’s skate scene  

---

## ⭐ Core Features (MVP)
1. **Destination Card Grid**  
   Responsive gallery of skate parks with images and quick info.

2. **Filter & Search**  
   Filter by country/city + keyword search.

3. **Park Detail Modal**  
   Difficulty level, amenities, photos, and local tips.

4. **Favourites List**  
   Save parks using LocalStorage.

5. **Ratings & Reviews**  
   Community-driven feedback system.

---

## 🧱 Project Structure (Planned)
